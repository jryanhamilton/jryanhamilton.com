JRyanHamilton.com

This is my personal website. My goal for the site was to have a place where I could display information about myself, the work I have done, and the projects I am currently working on.

The website also provided me with an opportunity to try something different with the design and layout. In designing the site, I wanted the focus to be on content. I felt that using a minimalistic approach to the visible elements would make the content be more prominent. I also wanted each section of the site to be visually identifiable, while keeping the minimalistic design. To accomplish this, I used colors to identify the various portions of the site.

Link: www.jryanhamilton.com

Utah Valley University Virtual Tour

For our senior project, my group designed and created the UVU Virtual Tour for Utah Valley University. It is a Adobe Flex based web application that uses XML driven content to load the text, and many of the images, and a CSS stylesheet to style the MXML components.

My specific responsibility was in creating the functionality and animations for the initial map portion.

Link: www.uvu.edu/tour

Zen Garden (School Project)

Just about every web designer or developer has heard about the Zen Garden project (www.csszengarden.com). It is a website that is based around the idea of multiple designers or developers creating various CSS stylesheets to change the look of the original zen garden HTML.

This was the goals of one of my school projects, where our teacher provided the HTML and we had to create the CSS stylesheet and accompanying graphics.
